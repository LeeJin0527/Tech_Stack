# 타임리프 (Thymeleaf) 
## 스프링에서의 화면 그리는 방식 
1. 스프링 어플리케이션이 템플릿 엔진을 가지고 뷰에 타임리프 문법이 있는지 확인.
2. 타임리프 문법을 이용해 html 생성 
3. 웹브라우저에서 **서버사이드 렌더링**을 통해 화면 그림
